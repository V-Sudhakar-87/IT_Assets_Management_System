💻 IT Asset Management System

A full-stack IT Asset Management System built to manage company assets, employee allocations, and maintenance requests efficiently. The system provides separate dashboards for administrators and employees with real-time asset tracking and management features.

🚀 Features

🔐 Authentication System
Secure Admin & Employee Login
Password Encryption using Bcrypt
Role-Based Access Control
Admin Registration with Secret Key Protection

👨‍💼 Admin Dashboard
Interactive Dashboard Overview
Asset Statistics & Reports
Employee Management
Asset Management
Asset Allocation Tracking
Maintenance Request Monitoring
System Settings Management
Analytics & Charts

👨‍💻 Employee Dashboard
View Assigned Assets
Raise Maintenance Requests
Track Request Status
Manage Profile Information
Change Password Feature

📦 Asset Management
Add New Assets
Edit Asset Information
Delete Assets
Asset Availability Tracking
Asset Status Monitoring
Warranty & Purchase Tracking

🔄 Asset Allocation
Allocate Multiple Assets to Employees
Prevent Duplicate Allocations
Allocation History Tracking
Asset Status Auto Updates

🛠 Maintenance Module
Raise Maintenance Requests
Priority-Based Request Management
Admin Response System
Request Status Tracking

📊 Reports & Analytics
Asset Category Distribution
Allocation Statistics
Maintenance Overview
Dashboard Charts using Chart.js
